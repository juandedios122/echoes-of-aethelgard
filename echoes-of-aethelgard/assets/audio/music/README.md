# Música del Juego

Coloca aquí los archivos de música en formato MP3 u OGG.

## Archivos necesarios:

1. **menu_theme.mp3** - Música del menú principal
2. **battle_theme.mp3** - Música de batalla
3. **gacha_theme.mp3** - Música para la pantalla de invocaciones
4. **exploration_theme.mp3** - Música de exploración

## Sitios recomendados para descargar música libre:

### 1. Incompetech (Kevin MacLeod)
- URL: https://incompetech.com/music/royalty-free/music.html
- Busca: "Medieval", "Epic", "Battle"
- Recomendados:
  - "Crusade" (batalla)
  - "Enchanted Journey" (exploración)
  - "Heroic Age" (menú)

### 2. FreePD (Dominio Público)
- URL: https://freepd.com/
- Categorías: Epic, Orchestral, Medieval

### 3. OpenGameArt
- URL: https://opengameart.org/
- Filtrar por: Music, Medieval/Fantasy

### 4. Pixabay
- URL: https://pixabay.com/music/
- Busca: "epic medieval", "fantasy battle"

## Cómo usar:

1. Descarga la música de los sitios anteriores
2. Renombra los archivos según la lista de arriba
3. Colócalos en esta carpeta (assets/audio/music/)
4. Godot los importará automáticamente

## Formato recomendado:
- MP3 para música (menor tamaño)
- OGG para mejor calidad
