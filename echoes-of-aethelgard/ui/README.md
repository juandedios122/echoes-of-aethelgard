# Carpeta UI - Recursos de Interfaz

## Nota Importante

La textura de fondo de las fichas de héroe se encuentra en:
`res://assets/ui/hero_detail_bg.png.jpeg`

Para reemplazarla con tu nueva textura de pergamino oscuro con bordes metálicos:
1. Reemplaza el archivo `assets/ui/hero_detail_bg.png.jpeg` con tu nueva imagen
2. O renombra tu nueva imagen a `hero_detail_bg.png.jpeg` y colócala en `assets/ui/`

**Especificaciones recomendadas:**
- Formato: PNG o JPEG
- Tamaño recomendado: 256x256 píxeles o superior
- Estilo: Pergamino oscuro con bordes metálicos
- El sistema aplicará automáticamente un tinte según la rareza del héroe
